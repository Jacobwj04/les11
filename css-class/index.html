hier komt cheat

Command cd

Gebruik je om van map te wisselen

Command git init

gebruik je om een niuwe repository aan te maken.

Command git add .

Gebruik je om als wat in dat folder dit toetevroegen aan je repository.

Command git commit -m "naam van commit"

Een commit gebruik je om je bestanden opteslaan in een repository.

Maak een repository op github

Je gebruik een repository op git hub om je bestanden te delen met anderen maar ook om makelijk terug te vinden.

Command git remote add origin (URL)

Je gruikt dat om je repository van je folder te verbinden met de repository op github.

Command git branch -M main

gebruik je aan te geven dat je de main branch gaat pushen naar github.

Command git push -u origin main

Gebruik je om als dat je heb geadd uploaden op github.
